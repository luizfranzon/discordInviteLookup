<!-- GETTING STARTED -->
## Getting Started

**1- Clone this repository:**
```bash
git clone https://github.com/luizfranzon/discordInviteLookup.git
```
**2- Install dependencies:**
```bash
npm install
```
Or
```bash
yarn
```
**3- Run the server:**
```bash
node SERVER.js
```

## Usage

Get a discord link and paste it in the input field.
